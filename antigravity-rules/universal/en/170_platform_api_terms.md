# 170. Platform API Terms & Compliance

## 1. Google Ecosystem
*   **Google Maps Platform**:
    *   **No Caching**: Storing or caching map data (except for lat/long and Place IDs) is strictly limited by the ToS. Unless for temporary caching (e.g., < 30 days), fetch from the API every time.
    *   **Attribution**: Hiding or modifying the Google logo and copyright notice is prohibited.
*   **YouTube Data API & Player**:
    *   **No Background Play**: Implementing "background playback" (a Premium feature) via the API is a violation.
    *   **No Separation**: Separating audio from video (audio-only playback) is prohibited.
*   **AdMob / AdSense**:
    *   **Invalid Traffic**: Clicking your own ads or encouraging users to click (Incentivized Clicks) results in an immediate ban. Always use "Test Ad IDs" during development.

## 2. Apple Ecosystem
*   **Sign in with Apple**:
    *   **Mandate**: If you implement third-party logins like Google or Facebook, you MUST implement **Sign in with Apple** with equal prominence (size, position) as per guidelines.
*   **Apple Pay**:
    *   **UI Guidelines**: There are strict rules for the Apple Pay button size, color, and surrounding whitespace. Do not style it freely with CSS; use the official assets or API.

## 3. Branding Guidelines
*   **Store Badges**:
    *   Use the official assets for "Get it on Google Play" and "Download on the App Store" exactly as provided. No color changes, aspect ratio distortion, or excessive decoration.
    *   **Clear Space**: Ensure the required clear space around the badges.

## 4. Cloud & AI Services
*   **OpenAI / LLM APIs**:
    *   **Content Moderation**: When using Generative AI, you are obligated to use tools like OpenAI's **Moderation Endpoint** to block inappropriate content (Hate, Violence, Sexual).
    *   **AI Disclosure**: Notify users that the content is generated by AI.
*   **AUP (Acceptable Use Policy)**:
    *   **AWS / GCP**: Adhere to the Cloud Provider's AUP. Unauthorized port scanning, crypto mining, and spamming result in immediate account suspension.
