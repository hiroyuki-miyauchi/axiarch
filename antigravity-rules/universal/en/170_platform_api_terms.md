# 170. Platform API Terms & Compliance

## 1. Google Ecosystem
*   **Google Maps Platform**:
    *   **No Caching**: Storing or caching map data (except for lat/long and place IDs) is strictly restricted by the Terms of Service. Except for temporary caching (e.g., within 30 days), fetch from the API every time in principle.
    *   **Attribution**: Hiding or altering the Google logo and copyright notice is prohibited.
*   **YouTube Data API & Player**:
    *   **No Background Play**: Implementing "background playback", a feature for YouTube Premium members, via API is a violation of terms.
    *   **No Separation**: Separating video and audio playback (e.g., audio-only playback) is prohibited.
*   **AdMob / AdSense**:
    *   **Invalid Traffic**: Developers clicking their own ads or asking/encouraging users to click (incentivized clicks) will result in immediate account suspension (BAN). Always use "Test Ad IDs" during testing.

## 2. Apple Ecosystem
*   **Sign in with Apple**:
    *   **Mandate**: If implementing third-party authentication like Google Login or Facebook Login, guidelines mandate implementing **Sign in with Apple** with equal prominence (size, position).
*   **Apple Pay**:
    *   **UI Guidelines**: There are strict regulations for the size, color, and surrounding margins of the Apple Pay button. Do not arbitrarily change styles with CSS; use official assets or APIs provided by Apple.

## 3. Branding Guidelines
*   **Store Badges**:
    *   Use official assets provided by each company for "Get it on Google Play" and "Download on the App Store" badges as is. Color changes, aspect ratio changes, and excessive decoration are prohibited.
    *   **Clear Space**: Ensure specified margins (clear space) around the badges.

## 4. Cloud & AI Services
*   **OpenAI / LLM APIs**:
    *   **Content Moderation**: When users use Generative AI, there is an obligation to block inappropriate content (hate, violence, sexual expression) using OpenAI's **Moderation Endpoint** or similar.
    *   **AI Disclosure**: Notify users that the content is generated by AI.
*   **AUP (Acceptable Use Policy)**:
    *   **AWS / GCP**: Comply with the Cloud Provider's AUP. Unauthorized port scanning, crypto mining, and spamming are subject to immediate account suspension.
